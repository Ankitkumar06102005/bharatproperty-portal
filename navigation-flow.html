🔧 How to Test

  1. Start with index.html - This is your main entry point
  2. Test registration - Go to login.html and create an account
  3. Check OTP - Look in the browser console for the OTP code
  4. Access dashboard - After OTP verification, you'll be redirected
  5. Navigate freely - Use the navigation links to move between pages

⚠️ Important Notes

  • Back button works - Browser back button will take you to previous pages
  • OTP is shown in console - Press F12 → Console to see the OTP code
  • Navigation is consistent - All pages have links back to home
  • Authentication required - Dashboard only accessible after login
